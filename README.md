# Building-a-Blackjack-Game
This is a simple implementation of the classic casino card game Blackjack using HTML, CSS, and JavaScript. The game allows players to play against the computerized dealer and test their luck and strategy in reaching a hand value as close to 21 as possible without going over.
Features:

Interactive user interface with a visually appealing design.
Play against the computerized dealer.
Player actions include hitting (drawing a card) and standing (keeping the current hand).
Dealer follows standard house rules for drawing cards.
The game tracks and displays the player's current hand value.
Win, lose, and push conditions are checked and displayed after each round.
Option to play again and start a new round.
